SQL atau Standard Query Language adalah bahasa pemrograman yang digunakan dalam
mengakses, mengubah, dan memanipulasi data yang berbasis relasional

Fungsi SQL:
  1. mengakses maupun mengubah database
  2. menjalankan sebuah query maupun mengambil data yang dibutuhkan
  3. memperbarui atau menyisipkan data dalam database
  4. menghapus sebuah data pada database
